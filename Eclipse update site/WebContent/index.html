Welcome to the Arduino eclipse plugin site

This page is about a free eclipse plugin to make development of big Arduino projects easier. As you only need the Arduino IDE, Eclipe and the plugin this Arduino IDE alternative is completely free and open source.

The plugin suppports

  • Creation of an Arduino sketch
  • Import of an Arduino Library
  • Change the Arduino board
  • Change the com port
  • Upload to the board
  • Real time build
  • User selected build.

To use it you only need:

  • the Arduino IDE
  • Eclipse Indigo (C/C++ version)
  • this plugin

Before continuing Note that :

  • The current version works with cpp files only (no pde or ino out of the box). Therefore the plugin is not compatible with the IDE out of the box. Below is a link to instructions on how to keep your code Arduino IDE compatible. I use those and I can compile the code between the Arduino IDE and eclipse without changes.
  • Note that the current version (1.2.0 ) works on windows and Linux and OS X.
  • There is currently no serial monitor in this project. Use the IDE one or any one of your choice
  • There is USB search (the Arduino connects through USB) in the tool.
  • Read the faq on the link between this tool and the WINAVR eclipse plugin.
